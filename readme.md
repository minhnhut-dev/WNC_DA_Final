
# Các công nghệ và thư viện sử dụng trong Project

- ReactJS
- MapBox (Load map và xử lý geocode)
- NestJS
- Leaflet (Hỗ trợ các custom cho marker)

# Setup
1. Di chuyển đến project
```
cd /project-name
```
2. cài node_modules (cài nodejs trước)
```
npm install
```
3. Chạy Project
```
npm start
```
